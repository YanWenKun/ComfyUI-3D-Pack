ComfyUI-3D-Pack for Comfy3D-WinPortable
---------------------------------------

This is a fork of https://github.com/MrForExample/ComfyUI-3D-Pack
for https://github.com/YanWenKun/Comfy3D-WinPortable
and https://github.com/YanWenKun/ComfyUI-Docker/tree/main/comfy3d-pt25

I removed the install scripts and requirements files to make sure
ComfyUI-Manager won't update this custom node and its dependency.

Please do not use this repository unless you are confident in your
understanding of its contents. Otherwise, it is recommended to use
the original ComfyUI-3D-Pack.
